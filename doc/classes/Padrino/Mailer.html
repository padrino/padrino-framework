This component uses the ‘mail’ library to create a powerful but simple mailer system within Padrino (and Sinatra). There is full support for using plain or html content types as well as for attaching files. The MailerPlugin has many similarities to ActionMailer but is much lighterweight and (arguably) easier to use.
